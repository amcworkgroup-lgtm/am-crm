# 🛠 AM Store CRM — Покрокова інструкція розгортання

## Що ви отримуєте
Повноцінна веб-CRM яка:
- Працює в браузері на будь-якому пристрої (телефон, планшет, ПК)
- Зберігає всі дані в базі даних на сервері
- Захищена паролем
- Має: облік ремонтів, статуси, фінанси, акти, майстри

---

## ВАРІАНТ А — Безкоштовний хостинг (Railway.app) ✅ Рекомендовано

### Крок 1 — Підготовка файлів
1. Створіть папку `am-crm` на комп'ютері
2. Помістіть в неї всі файли з цього архіву:
   ```
   am-crm/
   ├── server.js
   ├── package.json
   ├── public/
   │   └── index.html
   └── data/         ← створіть порожню папку
   ```

### Крок 2 — GitHub (безкоштовно)
1. Зареєструйтесь на **github.com**
2. Натисніть **"New repository"** → назвіть `am-crm` → **Create**
3. Завантажте свої файли (кнопка **"uploading an existing file"**)
4. Натисніть **Commit changes**

### Крок 3 — Railway (безкоштовний хостинг)
1. Зайдіть на **railway.app** → **"Start a New Project"**
2. Оберіть **"Deploy from GitHub repo"**
3. Підключіть свій GitHub і оберіть репозиторій `am-crm`
4. Railway автоматично запустить сервер!

### Крок 4 — Налаштування пароля
1. В Railway → ваш проєкт → **Variables**
2. Додайте змінну:
   ```
   ADMIN_PASSWORD = ВашПарольТут
   ```
3. Натисніть **Deploy** — готово!

### Крок 5 — Отримати посилання
- В Railway → вкладка **Settings** → **Domains** → **Generate Domain**
- Ви отримаєте посилання типу: `https://am-crm-production.up.railway.app`
- Це і є адреса вашої CRM — відкривайте з будь-якого пристрою!

## ВАРІАНТ Б — VPS сервер (~5$/місяць, більше контролю)

### Крок 1 — Орендуйте сервер
Оберіть одне з:
- **Hetzner** (hetzner.com) — CX11, ~4€/міс, найдешевше
- **DigitalOcean** — $6/міс
- **Cityhost** (cityhost.ua) — ~150 грн/міс (Україна)

Оберіть: **Ubuntu 22.04**, найменший тариф.

### Крок 2 — Підключення до сервера
На Windows — завантажте **PuTTY** (putty.org)

Підключіться:
```
Host: IP-адреса_вашого_сервера
Port: 22
Login: root
```

### Крок 3 — Встановлення Node.js
Вставте команди по черзі:
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
node --version   # має показати v20.x.x
```

### Крок 4 — Завантаження файлів CRM
```bash
mkdir -p /var/www/am-crm/public /var/www/am-crm/data
cd /var/www/am-crm
```

Завантажте файли через **WinSCP** (winscp.net) — безкоштовна програма.
Підключіться так само як PuTTY і перетягніть файли в папку `/var/www/am-crm/`

### Крок 5 — Встановлення залежностей
```bash
cd /var/www/am-crm
npm install
```

### Крок 6 — Налаштування пароля
```bash
nano .env
```
Напишіть:
```
ADMIN_PASSWORD=ВашПарольТут
PORT=3000
```
Збережіть: Ctrl+O → Enter → Ctrl+X

### Крок 7 — Запуск CRM (автоматичний рестарт)
```bash
npm install -g pm2
pm2 start server.js --name am-crm
pm2 startup
pm2 save
```

### Крок 8 — Відкрити порт (щоб зайти з браузера)
```bash
sudo ufw allow 3000
sudo ufw enable
```

Тепер CRM доступна за адресою:
```
http://IP_ВАШОГО_СЕРВЕРА:3000
```

### Крок 9 — Красивий домен (необов'язково)
1. Купіть домен на **nic.ua** (~150 грн/рік)
2. Встановіть Nginx:
```bash
sudo apt install nginx
sudo nano /etc/nginx/sites-available/am-crm
```
Вставте:
```nginx
server {
    listen 80;
    server_name ваш-домен.ua;
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
    }
}
```
```bash
sudo ln -s /etc/nginx/sites-available/am-crm /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx
```

## Управління CRM

### Резервна копія бази даних
```bash
cp /var/www/am-crm/data/crm.db /var/backups/crm_$(date +%Y%m%d).db
```

### Оновлення файлів
Завантажте нові файли через WinSCP і виконайте:
```bash
pm2 restart am-crm
```

### Перегляд логів (якщо щось не працює)
```bash
pm2 logs am-crm
```

## Часті питання

**❓ CRM не відкривається**
→ Перевірте що pm2 запущено: `pm2 status`
→ Перевірте порт: `sudo ufw status`

**❓ Забули пароль**
→ Відредагуйте `.env` файл і перезапустіть: `pm2 restart am-crm`

**❓ Як зайти з телефону**
→ Просто відкрийте браузер і введіть адресу CRM — інтерфейс адаптований під мобільний
